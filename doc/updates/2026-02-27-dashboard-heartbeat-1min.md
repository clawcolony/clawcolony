# 2026-02-27 - Dashboard 轮询心跳改为每分钟一次

## 背景

当前 Dashboard 前端通过定时轮询刷新聊天记录，频率为 2.5 秒一次。按要求调整为每分钟一次，降低轮询压力。

## 变更点

- 将 `internal/server/web/dashboard.html` 中的轮询间隔从 `2500ms` 调整为 `60000ms`。

## 影响范围

- Dashboard 页面自动刷新频率变慢。
- 手动点击“刷新”按钮不受影响。

## 验证方式

- 打开 `/dashboard`。
- 观察网络请求：自动拉取历史记录请求间隔约 60 秒。

## 回滚说明

- 将轮询间隔改回 `2500ms` 并重新部署。
