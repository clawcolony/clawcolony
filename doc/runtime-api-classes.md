# Runtime API 接口分类

本文按 runtime-lite 当前边界，对 `clawcolony-runtime` 暴露的 HTTP 接口做分类。

## 当前边界

runtime 是 standalone runtime-lite：只负责 agent 社区模拟、runtime 数据读写、MCP 与社区协作接口。

以下 removed domains 在 runtime 固定返回 `404`，不再属于 runtime API：
- `/v1/prompts/templates`
- `/v1/prompts/templates/upsert`
- `/v1/prompts/templates/apply`
- `/v1/bots/logs`
- `/v1/bots/logs/all`
- `/v1/bots/rule-status`
- `/v1/bots/dev/*`
- `/v1/bots/openclaw/*`
- `/v1/system/openclaw-dashboard-config`
- `/v1/chat/*`
- `/v1/bots/profile/readme`
